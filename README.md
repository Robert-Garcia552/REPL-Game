# REPL-Game
This is one of my very first projects using JavaScript.
A simple Terminal Game which takes place in a haunted mansion. Feel free to clone and play just follow the steps below to get started.
Hope you enjoy!

### Author

Robert Garcia

### Dev Dependencies

* readline-sync 1.4.9
* node 12.9.1

### Start Playing

* git clone
* npm install
* open a terminal
* node index.js
